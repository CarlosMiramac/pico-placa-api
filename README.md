# API Pico y Placa

## Descripción

API REST desarrollada para consultar restricciones de pico y placa según el número de placa y el día.

##  Tecnologías

* Node.js
* NestJS
* PostgreSQL
* Jest / Supertest

##  Funcionalidades

* Consulta de restricción vehicular
* Lógica de rotación por día
* API REST estructurada

##  Endpoints

* GET /pico-placa/:placa
* GET /estado

##  Cómo ejecutar

```bash
npm install
npm run start
```

##  Pruebas

```bash
npm run test
```
